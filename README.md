# Bank Loan Analysis Dashboard 🏦

An end-to-end Bank Loan Analysis project using MySQL for data 
exploration and Power BI for interactive dashboard visualization.

## 🎯 Objective
Analyze bank loan data to calculate key KPIs, classify loans into 
good vs bad, perform trend analysis, and identify factors affecting 
loan approval and repayment.

## 🛠️ Tools Used
- MySQL — Data exploration & KPI queries
- Power BI Desktop — Dashboard & visualization
- Power Query — Data cleaning & transformation
- DAX — Custom measures & calculations

## 📌 Key Features
- 5 KPI Cards: Total Applications, Funded Amount, Amount Received, 
  Good Loan %, Bad Loan %
- Good vs Bad Loan classification (86.18% vs 13.82%)
- Monthly trend analysis (Jan–Dec)
- Top 7 Loan Purposes & Top 10 Loan States
- Loan by Grade (risk-based approval factors)
- Loan by Term (36 vs 60 months)
- Interactive slicers: Verification Status + Loan Category

## 🔍 Key Insights
- 86.18% of loans are good (Fully Paid/Current)
- Debt consolidation is the top loan purpose (18K applications)
- California leads all states with 7K loan applications
- Grade B has the highest number of applications (11.7K)
- 73.2% of loans are 36-month term
- Total amount received ($473M) exceeds funded amount ($436M)
